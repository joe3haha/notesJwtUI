Notes

This page contains a list of notes that are persisted in your local instance of mongoDB by the spring backend that goes along with this angular front-end.

If you click on one of the notes you will be navigated to a new component view which allows the editing of the note. It also shows the message associated with the note's title

If you click the plus sign next to the note header above you will be navigated to another component view where you can create a new note that will be reflected on this list.

Hitting the delete (trashcan) icon will send a delete request to the spring backend that will remove the note from the database
