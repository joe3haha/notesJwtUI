Notes - About this project

This is the second iteration of my note taking app's front-end. I'm using angular 4.2 for this particular project. This front-end is associated with the following back-end: Spring boot back-end. The purpose of this project was two fold; To build my portfolio of projects on github and to learn Angular.

This project allows users to create and manipulate a list of notes. Users can create / edit / delete these notes through this UI. I used W3.CSS instead of creating my own custom CSS. It made sense to just utilize this pre-existing CSS so that I could focus on learning Angular. Here is a link to W3.CSS. More specifically I utilized and edited one of the templates provided: this one . Shoutout to W3 for this!

This project uses JWT for authentication against the stateless backend server. Once you login a token will be stored in local storage and sent with any request made against the back-end. This is done via an interceptor (really cool in angular) that appends any request with the appropriate headers

I protected the routes using a service that checks the expiration time of the JWT and either redirects to the login page or allows a user to navigate to the appropriate page.

The navigation pane will show different icons to navigate to depending on the authenticated state of the user. I used observables and a service to keep track of authenticated state of a user.

Go ahead and create a new user if you like, or use the default accounts I created whenever the back-end is started. un: pojo - pw: test or un: wiggle - pw: test

The pojo account will have a single note already generated. Feel free to contact me with any questions.
